# Notes Organization

This server setup allows you to store off (vaulting) web pages, PDFs, images, notes, etc that you want to save and search later.  On top of that there is a reference manager running to help manage citions, metadata and organizing your literature for research purposes.  Finally, there is a notworked notes tool to allow you to build "thinking spaces", to link ideas, drafting, creating outlines, etc.
